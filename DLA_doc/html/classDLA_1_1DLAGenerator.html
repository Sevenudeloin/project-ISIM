void DLA::DLAGenerator::generateHeightmaps ( Heightmap &  base_heightmap,
Heightmap &  upscaled_heightmap

Generate the base heightmap (for the mesh) and the upscaled heightmap (for the texture map and the normal map). The upscaled heightmap will be generated by the DLA algorithm. Base heightmap is a downsampled version of the upscaled heightmap.

Parameters
[out]base_heightmapbase heightmap (for the mesh), width must be a power of 2.
[out]upscaled_heightmapupscaled heightmap (for the texture map and the normal map), width must be a power of 2 (>= 2^4).
